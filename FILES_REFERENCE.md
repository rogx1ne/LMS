# LMS - Complete Files Reference

## Essential Files for Users

### Setup (One-time)
- **`LMS-Setup.jar`** - Setup wizard (run once to initialize database)

### Running the Application (After Setup)

#### Windows
- **`run.bat`** - Double-click to launch application ✨
- **`create-shortcut.bat`** - Double-click to create Desktop icon

#### Linux/Mac
- **`run.sh`** - Double-click to launch application ✨
- **`create-shortcut.sh`** - Run to create Desktop icon

### Documentation (Read if needed)
- **`APPLICATION_LAUNCH_GUIDE.md`** - How to launch the app (user-friendly)
- **`LAUNCHER_GUIDE.md`** - Technical launcher configuration
- **`README.md`** - Project overview
- **`INSTALLATION_GUIDE.md`** - Detailed setup instructions

### Database (For troubleshooting)
- **`cleanup_prj2531h.sql`** - Reset database (if you need to reinstall)
- **`script.sql`** - Database schema (created by setup wizard)

### Application Files (Auto-created by setup)
- **`bin/`** - Compiled application classes
- **`lib/`** - Required Java libraries
- **`src/`** - Source code

## Quick Start by Platform

### Windows Users
```
1. Run:        java -jar LMS-Setup.jar
2. After:      Double-click run.bat
3. Or:         Double-click create-shortcut.bat for Desktop icon
```

### Linux/Mac Users
```
1. Run:        java -jar LMS-Setup.jar
2. After:      ./run.sh
3. Or:         ./create-shortcut.sh for Desktop icon
```

## For Support

If something doesn't work:

1. **Read**: `APPLICATION_LAUNCH_GUIDE.md` (user-friendly)
2. **Check**: `LAUNCHER_GUIDE.md` (technical details)
3. **Verify**: Oracle database is running on localhost:1521
4. **Test**: Run setup again: `java -jar LMS-Setup.jar`

## What Users Should NOT Touch

❌ `src/` - Source code (don't modify)
❌ `bin/` - Compiled classes (auto-generated)
❌ `lib/` - Library files (don't modify)
❌ `script.sql` - Database schema (don't modify)

## What Users CAN Modify

✅ `run.bat` / `run.sh` - Change database URL if needed
✅ `create-shortcut.*` - Optional, for Desktop access
✅ Configuration via environment variables
